# 杜家瑞 (CloverCLC) — 个人简历网站

基于纯 HTML/CSS/JS 构建的个人简历页面，部署在 GitHub Pages。

## 关于我

2027 届毕业生，就读移动应用开发专业。专注于 AI Agent 开发方向，熟悉 pydantic_ai、LangChain、LangGraph 等框架，具备全栈开发能力。

## 技术栈

- **AI Agent:** pydantic_ai · LangChain · LangGraph · MCP 协议
- **前端:** Nuxt · Vue.js
- **后端:** Python · WebSocket · REST API
- **语言:** TypeScript · Python · Java · C

## 项目展示

- **[26hw_ai](https://github.com/CloverCLC/26hw_ai.git)** — 支持自定义模型和 MCP 工具的聊天 Agent，前端 Nuxt + 后端 pydantic_ai
- **[Chatroom](https://github.com/CloverCLC/chatroom.git)** — 基于 WebSocket 的实时聊天室，pydantic_ai + Python 后端

## 本地运行

```bash
git clone https://github.com/CloverCLC/cloverclc.github.io.git
cd cloverclc.github.io
# 直接用浏览器打开 index.html，或用任意静态服务器
python -m http.server 8080
```

打开 http://localhost:8080 查看。
